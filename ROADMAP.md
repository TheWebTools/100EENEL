# 100EENEL Roadmap

This roadmap outlines the planned evolution of 100EENEL from a local log‑monitoring agent into a full multi‑platform security observability system.

---

## ✅ v0.2.0 — Core Enhancements (Next Release)
- Alert history storage (SQLite)
- Trend graph API endpoints
- Rule testing endpoint
- Alert cooldowns (anti‑spam)
- Multi‑condition rule matching
- Configurable polling interval
- Improved log parsers (Windows + Linux)

---

## 🚧 v0.3.0 — UI & Dashboard
- MAUI Desktop App (Windows + Linux)
- MAUI Mobile App (Android + iOS)
- Real‑time log viewer
- Trend graphs (errors/hour, warnings/day)
- Alert timeline visualization
- Settings UI for alert rules
- Local agent status dashboard

---

## 🌐 v0.4.0 — Cloud & Remote Monitoring
- Cloud dashboard (optional SaaS)
- Remote agent registration
- Multi‑server monitoring
- Centralized alert management
- Cloud‑based trend analytics
- Secure API tokens for agents

---

## 🔌 v0.5.0 — Integrations
- Webhooks for custom integrations
- Microsoft Teams integration
- PagerDuty integration
- Syslog ingestion
- File‑based log ingestion

---

## 🛡️ v1.0.0 — Stable Release
- Full documentation
- Installers for Windows & Linux
- Docker image
- Production‑ready builds
- Security hardening
- Automated CI/CD pipeline
